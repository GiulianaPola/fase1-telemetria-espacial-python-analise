# fase1-telemetria-espacial-python-analise
Este repositório contém a documentação e os algoritmos desenvolvidos para a Atividade Integradora da Fase 1 da FIAP. O projeto consiste em um sistema de verificação de prontidão para lançamento espacial, analisando telemetria em tempo real para decidir entre a decolagem ou o aborto da missão.
